Help for ChEMBL wizard

( If the wizard is not open yet open it by right clicking on a project and choose chembl logo.)

1. Depending on what perspective to search from select 'compound' or 'target' (1).

2. In the text field you are able to search with: (2)

If 'compounds' is checked:

chebi id

SMILES (slow)

keyword

If 'target' is checked:

target id

protein sequence

keyword

3. As you type the most left table will fill.

4.The right most table is for saving. To save data it has to be draged to this table. This can be done by checking the boxes in the left table or drag and drop. If you wich to save all simply press the button 'Save all'. (4)

5.The data is save in to a csv file.

6. If to remove data in the right table check the box and push the button 'remove'.
